# 🚀 QUICK START - Система аккаунтов

## ⚡ За 30 секунд

```bash
# Терминал 1 - WebSocket сервер
node websocket-server.js

# Терминал 2 - HTTP сервер  
node server.js

# Браузер
http://localhost:3000
```

## 🧪 Тестирование (2 минуты)

### 1. Регистрация
- Нажмите "Регистрация"
- Введите: test / test123
- Нажмите "Зарегистрироваться"

### 2. Игра
- Нажите на косатку (запомните количество монет)

### 3. Проверка сохранения
- Нажмите F5 (перезагрузка)
- Введите пароль заново
- Монеты должны остаться такие же ✅

## ✅ Если все сработало - готово!

## ❌ Если не работает

**Проблема:** "Ошибка подключения к серверу"
- **Решение:** Запустить WebSocket сервер

**Проблема:** "Неверный пароль"
- **Решение:** Очистить браузер (F12 → Application → Clear storage)

**Проблема:** Данные не сохраняются
- **Решение:** Подождать 30 сек или смотреть GUIDE.md раздел "Отладка"

## 📊 Что происходит

```
РЕГИСТРАЦИЯ:
  Форма → WebSocket → Сервер → БД → Ответ → Игра

СОХРАНЕНИЕ (каждые 30 сек):
  Игра → WebSocket → Сервер → БД (database.json)

ПЕРЕЗАГРУЗКА (F5):
  Браузер → localStorage → WebSocket → Сервер → БД → Игра
```
