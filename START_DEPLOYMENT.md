## ⚡ БЫСТРАЯ СПРАВКА

### Что нужно

- ✅ GitHub аккаунт (бесплатно)
- ✅ Render аккаунт (бесплатно)
- ✅ Vercel аккаунт (бесплатно)
- ✅ Git установлен локально

### Этапы (4 шага)

1. **GitHub** (1 мин) - `git push` ваш код
2. **Render** (2 мин) - создайте Web Service
3. **Vercel** (2 мин) - импортируйте репозиторий
4. **Финиш** (1 мин) - обновите переменные окружения

**Итого:** 5-7 минут работы

## ✅ ПРЕ-ДЕПЛОЙМЕНТ ПРОВЕРКА

Перед началом убедитесь что все готово:

**На Windows:**
```bash
scripts\check-deployment.bat
```

**На Mac/Linux:**
```bash
chmod +x scripts/check-deployment.sh
./scripts/check-deployment.sh
```

Это проверит:
- ✅ Node.js и npm установлены
- ✅ Все файлы на месте
- ✅ Нет синтаксических ошибок
- ✅ Git инициализирован
- ✅ Готово к деплойменту!
