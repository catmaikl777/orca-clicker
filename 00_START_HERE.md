## 💻 Архитектура

```
User Input (Browser)
    ↓
auth-client.js (Login UI)
    ↓ WebSocket
client.js (Game Logic)
    ↓ WebSocket
websocket-server.js (Server)
    ↓ imports
auth.js (Account Logic)
    ↓ read/write
database.json (Database)
```

---

## 🔄 Основные процессы

### Регистрация/Вход
```
Browser form → authRequest → Server auth.js → 
Check/Create account → accountId + gameData → 
Browser localStorage → Game loaded ✅
```

### Сохранение прогресса
```
Game (30s interval) → savePlayerData → 
Server updates db.players → 
Save to database.json ✅
```

### Восстановление сессии
```
F5 reload → localStorage restore → 
restoreSession → Server loads from DB → 
Game continues ✅
```

## 📞 Решение типичных проблем

| Проблема | Решение |
|----------|---------|
| "Ошибка подключения" | Запустить websocket-server.js |
| "Неверный пароль" | Очистить браузер или перезагрузить |
| "Данные не сохраняются" | Подождать 30 сек или закрыть/открыть |
