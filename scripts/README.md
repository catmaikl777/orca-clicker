# 🗑️ Скрипты очистки базы данных

## Описание

Скрипты для полной очистки базы данных перед выпуском в продакшен.

## Скрипты

### clear-json-db.js
Очищает `database.json` (локальная база данных).

**Запуск:**
```bash
npm run clear-db
```

Или напрямую:
```bash
node scripts/clear-json-db.js --force
```

### clear-postgres-db.js
Очищает PostgreSQL базу данных (для продакшена).

**Требования:**
- Установлена переменная `DATABASE_URL` в `.env`
- `NODE_ENV=production`

**Запуск:**
```bash
npm run clear-postgres
```

Или напрямую:
```bash
node scripts/clear-postgres-db.js --force
```

## Что удаляется

### players
Все записи игроков (прогресс, монеты, скины и т.д.)

### clans  
Все кланы и их участники

### leaderboard
История лидерборда

### event_coins
Билеты ивента

### accounts
Все аккаунты пользователей

## Что сохраняется

- Структура таблиц (PostgreSQL)
- Конфигурация ивента (даты, награды)
- Статистика (сбрасывается на 0)

## ⚠️ ВАЖНО

1. **Резервное копирование**: Перед очисткой создайте бэкап
   ```bash
   npm run backup
   ```

2. **Подтверждение**: Скрипты требуют флаг `--force` для защиты от случайной очистки

3. **Невозможно отменить**: После очистки данные невозможно восстановить

## Пример использования

### Локальная разработка
```bash
# Очистить локальную БД
npm run clear-db

# Перезапустить сервер
npm run dev
```

### Продакшен (Render)
```bash
# Добавить переменную окружения в Render
DATABASE_URL=postgresql://...

# Очистить через SSH или локально
node scripts/clear-postgres-db.js --force
```

## Проверка после очистки

После очистки проверьте что:
1. Сервер запускается без ошибок
2. Можно создать новый аккаунт
3. Новый игрок имеет стартовые значения (0 монет, normal скин)
