# 🚀 Загрузка на Яндекс Игры

## 📋 Подготовка

### 1. Структура файлов
```
orca-clicker/
├── index.html
├── style.css
├── client.js
├── auth-client.js
├── yandex-games.js          # ✅ Новый файл - Яндекс SDK
├── yandex-games/
│   └── manifest.yaml        # ✅ Манифест приложения
├── icon.png                 # Иконка 512x512
├── screenshots/             # Скриншоты игры
└── build/                   # Сборка для публикации
```

### 2. Установка SDK
SDK Яндекс Игр уже подключен в `index.html`:
```html
<script src="https://yandex.ru/games/sdk/v2"></script>
<script src="yandex-games.js"></script>
```

## 🎮 Функции Яндекс Игр

### Реклама с наградой
- **Кнопка в меню**: 📺 Награда за реклама
- **Кнопка в настройках**: 📺 Получить награду за рекламу
- **Награда**: 1000 × уровень косаток
- **Ограничение**: можно смотреть каждые 30 минут (реализовать по желанию)

### Лидерборд
- Автоматическая инициализация при запуске
- Сохранение очков в облако Яндекс

### Облачное сохранение
- Синхронизация прогресса с облаком Яндекс
- Загрузка при входе в игру

## 📦 Сборка для публикации

### 1. Создайте архив
```bash
# Убедитесь, что все файлы на месте
cp index.html style.css client.js auth-client.js yandex-games.js build/
cp -r yandex-games/ build/
cp icon.png build/

# Создайте ZIP архив
cd build
zip -r ../orca-clicker-yandex.zip .
cd ..
```

### 2. Проверьте файлы
- ✅ `index.html` - главная страница
- ✅ `style.css` - стили
- ✅ `client.js` - логика игры
- ✅ `auth-client.js` - авторизация
- ✅ `yandex-games.js` - Яндекс SDK
- ✅ `icon.png` - иконка 512x512
- ✅ `manifest.yaml` - манифест

## 🌐 Публикация в Консоли Разработчика

### Шаг 1: Зарегистрируйтесь
1. Перейдите в [Консоль разработчика Яндекс Игр](https://games.yandex.ru/console/)
2. Войдите под Яндекс ID
3. Примите договор оферты

### Шаг 2: Создайте приложение
1. Нажмите "Добавить приложение"
2. Выберите "Новое приложение"
3. Заполните базовые данные:
   - **Название**: Косатка Клик! 🐋
   - **Описание**: Залипательный кликер с косатками!
   - **Категория**: Casual / Arcade
   - **Язык**: Русский

### Шаг 3: Загрузите файлы
1. Перейдите во вкладку "Исходный код"
2. Загрузите ZIP архив `orca-clicker-yandex.zip`
3. Дождитесь проверки (автоматически, ~5 минут)

### Шаг 4: Настройте иконки и скриншоты
1. **Иконки**: 20x20, 64x64, 128x128, 256x256 px
2. **Скриншоты**: 
   - Портретная ориентация: 720x1280, 1280x2560
   - Горизонтальная: 1280x720 (опционально)

### Шаг 5: Заполните информацию
- **Описание**: подробное описание игры
- **Категории**: casual, arcade
- **Возрастной рейтинг**: 0+ (если нет насилия)
- **Контакты разработчика**: email для поддержки

### Шаг 6: Опубликуйте
1. Нажмите "Сохранить"
2. Отправьте на модерацию
3. Дождитесь одобрения (~1-3 дня)

## 🔧 Настройка манифеста

Отредактируйте `yandex-games/manifest.yaml`:

```yaml
manifest_version: 2
id: orca-clicker-game          # Уникальный ID
name:
  ru: "Косатка Клик! 🐋"
  en: "Orca Clicker 🐋"
description:
  ru: "Ваше описание на русском"
  en: "Description in English"
author:
  name: "Ваше имя/компания"
  url: "https://ваш-сайт.com"
version: "1.0.0"
```

## 🧪 Тестирование

### Локальное тестирование

### Тестирование в Яндекс

2. Получите тестовую ссылку
3. Проверьте все функции:
   - ✅ Реклама работает
   - ✅ Лидерборд сохраняется
   - ✅ Облачное сохранение работает

## 📊 Монетизация

### Настройка рекламы
- **Interstitial**: полноэкранная реклама (между уровнями)
- **Rewarded**: реклама с наградой (кнопка в меню)

### Рекомендации
- Не показывайте рекламу слишком часто
- Давайте игрокам выбор (кнопка "получить награду")
- Используйте rewarded видео для бонусов

## 🐛 Устранение проблем

### Реклама не показывается
- Проверьте, что игра загружена на Яндекс
- Локально SDK может не работать - это нормально
- Проверьте консоль на ошибки

### Лидерборд не работает
- Убедитесь, что инициализация прошла успешно
- Проверьте права доступа в консоли
- Лидерборды нужно создать в консоли разработчика

### Облачное сохранение
- Убедитесь, что `cloud_save: true` в манифесте
- Проверьте, что Player инициализирован
