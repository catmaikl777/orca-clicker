## 📦 Команды для продакшена

### Очистить базу данных
```bash
# Локальная JSON БД
npm run clear-db

# PostgreSQL (для Render)
npm run clear-postgres
```

### Создать бэкап
```bash
npm run backup
```

### Запустить сервер
```bash
# Разработка
npm run dev

# Продакшен
npm start
```

## 🌐 Деплой на Render

### 1. Создать Web Service
- Connect GitHub repo
- Build Command: `npm install`
- Start Command: `npm start`

### 2. Добавить PostgreSQL
- Create New → PostgreSQL
- Получить DATABASE_URL

### 3. Настроить переменные окружения
```
NODE_ENV=production
DATABASE_URL=postgresql://...
PORT=3001
```

### 4. Очистить БД перед запуском
```bash
# Через Render Dashboard → Shell
node scripts/clear-postgres-db.js --force
```

## 🔧 Устранение проблем

### Ошибка подключения к БД
```
❌ PostgreSQL недоступен
```
**Решение:** Проверить DATABASE_URL в переменных окружения

### Сервер не запускается
```
❌ Port 3001 already in use
```
**Решение:** Установить PORT=3001 в переменных окружения

### Данные не сохраняются
```
❌ saveDB() error
```
**Решение:** Проверить права доступа к файлам (для JSON) или подключение к PostgreSQL

### Кэш браузера
```
⚠️ data is not defined
```
**Решение:** Очистить кэш (Ctrl+Shift+R) или использовать Hard Reload
