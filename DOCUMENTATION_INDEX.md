## 🔍 Быстрая справка

### Команды
```bash
node websocket-server.js    # Запуск WebSocket (порт 3001)
node server.js              # Запуск HTTP (порт 3000)
http://localhost:3000       # Открыть в браузере
```

### Логирование
```bash
# Браузер - F12 → Console
✅ Аутентификация успешна
💾 Данные сохранены
🔄 Сессия восстановлена

# Node.js - терминал
WebSocket сервер запущен
🆕 Регистрация
✅ Вход
```

### Проверка ошибок
```bash
# Браузер DevTools
F12 → Console (ищите ✅ и ❌)
F12 → Network → WS (смотрите соединение)

# Node.js консоль
Должны быть логи от сервера
```
