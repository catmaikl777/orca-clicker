## 🧪 Быстрые тесты

### ✅ Регистрация
1. Нажмите "Регистрация"
2. Введите: username = `test`, password = `test123`
3. Должны увидеть экран игры с вашим именем

### ✅ Сохранение прогресса
1. Поиграйте (нажмите на косатку)
2. Посмотрите монеты
3. Перезагрузите страницу (F5)
4. Введите пароль
5. Видите тот же прогресс

### ✅ Автовход после перезагрузки
1. Закройте браузер
2. Откройте заново
3. Должны **автоматически** войти без пароля

### ✅ Режим гостя
1. Нажмите "Играть без учета"
2. Поиграйте
3. Перезагрузите - прогресс не сохраняется

## 📊 Система работает так:

```
РЕГИСТРАЦИЯ / ВХОД:
  Browser → WebSocket → Server → auth.js → Database
  ↓
  Server отправляет accountId + gameData
  ↓
  Browser сохраняет в localStorage
  ↓
  Игра загружается с сохраненными данными

АВТОСОХРАНЕНИЕ (каждые 30 сек):
  Game progress → WebSocket → Server
  ↓
  Server обновляет database.json
  ↓
  Данные всегда актуальны

ПЕРЕЗАГРУЗКА СТРАНИЦЫ:
  Browser восстанавливает localStorage
  ↓
  Отправляет restoreSession на сервер
  ↓
  Server загружает данные из БД
  ↓
  Игра продолжается с того же места
```
