  - PostgreSQL адаптер готов
  - Автоматическое переключение режимов
  - Конфигурация для Render включена
  
- [x] **Rate limiting** ✅
  - Оптимизирован для игровых действий
  - 500 запросов/мин для кликов

  - 10 попыток входа/15 минут
  
- [x] **Backup стратегия** ✅
  - Автоматические бэкапы (CLI)
  - Сжатие gzip
  - Хранение 7 последних бэкапов

## 🚀 Быстрый старт

### 1. Установите зависимости:
```bash
npm install
```

### 2. Настройте переменные окружения:
```bash
# Копируйте .env.example в .env
cp .env.example .env

# Отредактируйте .env
nano .env
```

### 3. Запустите сервер:
```bash
# Разработка
npm run dev

# Продакшен
npm start

# С бэкапами
AUTO_BACKUP=true npm start
```

## 🔧 Команды бэкапов

```bash
# Создать бэкап
npm run backup create

# Показать список
npm run backup list

# Восстановить
npm run backup restore backups/backup-2024-01-01.json.gz

# Автосервис (24 часа)
npm run backup auto 24
```
