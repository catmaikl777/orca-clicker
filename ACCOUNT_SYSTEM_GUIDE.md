# 🎮 Инструкция по запуску системы аккаунтов

## 📌 Предварительные требования

- Node.js (v14+)
- npm пакеты установлены:
  - `ws` - WebSocket сервер
  - Остальные зависимости в `package.json`

## 🚀 Запуск

### 1️⃣ Установка зависимостей (если еще не установлены)
```bash
npm install
```

### 2️⃣ Запуск WebSocket сервера
```bash
node websocket-server.js
```
**Ожидаемый вывод:**
```
WebSocket сервер запущен на порту 3001
База данных загружена
🎉 Ивент активен! До конца: 20160 мин.
```

### 3️⃣ Запуск HTTP сервера (в другом терминале)
```bash
node server.js
```
**Ожидаемый вывод:**
```
HTTP сервер запущен на http://localhost:3000
```

### 4️⃣ Открыть игру в браузере
```
http://localhost:3000
```

## ✅ Проверка работы

### Тест 1: Регистрация нового аккаунта
1. Откройте http://localhost:3000
2. Нажмите "Регистрация"
3. Введите:
   - Имя: `test_player`
   - Пароль: `password123`
   - Подтверждение: `password123`
4. Нажмите "Зарегистрироваться"

**Что должно произойти:**
- ✅ Вы должны увидеть экран игры
- ✅ В верхнем правом углу должно появиться имя: "👤 test_player"
- ✅ В консоли браузера: `✅ Аутентификация успешна: test_player`
- ✅ В консоли Node.js: `🆕 Регистрация: test_player`

### Тест 2: Сохранение прогресса
1. После регистрации нажмите на косатку несколько раз
2. Посмотрите количество монет
3. **Закройте браузер** (или вкладку)
4. **Откройте игру заново**
5. **Введите свои учетные данные** и войдите

**Что должно произойти:**
- ✅ Видите той же экран входа (не автосохранение, нужен вход)
- ✅ Вводите username и пароль
- ✅ После входа видите такое же количество монет

### Тест 3: Автосохранение при перезагрузке (F5)
1. Войдите в аккаунт
2. Нажимайте на косатку несколько раз
3. Нажмите **F5** (перезагрузка страницы)
4. **Используйте восстановление сессии** (должно произойти автоматически)

**Что должно произойти:**
- ✅ Страница перезагружается
- ✅ В консоли: `✅ Автовход: test_player`
- ✅ Видите игру с тем же количеством монет
- ✅ Не требуется повторный ввод пароля

### Тест 4: Режим гостя
1. Нажмите "Играть без учета"
2. Нажимайте на косатку несколько раз
3. Перезагрузите страницу (F5)

**Что должно произойти:**
- ✅ Вернулись на экран входа
- ✅ Прогресс **не сохранился** (как и ожидается для гостя)

### Тест 5: Выход из аккаунта
1. Войдите в аккаунт, поиграйте
2. Нажмите кнопку "Выход" в верхнем левом углу
3. Подтвердите выход

**Что должно произойти:**
- ✅ Данные сохранены перед выходом
- ✅ Вернулись на экран входа
- ✅ В консоли: `👋 Вы вышли из аккаунта`

## 🔧 Типичные проблемы и решения

### ❌ "Ошибка подключения к серверу"

**Причина:** WebSocket сервер не запущен

**Решение:**
```bash
node websocket-server.js
```

### ❌ "Неверный пароль" при правильном пароле

**Причина:** Проблема с хешированием

**Решение:**
1. Проверьте что пароль содержит только буквы/цифры
2. Попробуйте пароль без спецсимволов
3. Очистите БД (`database.json`): `{ "accounts": {}, "players": {} }`

### ❌ Данные не сохраняются

**Причина:** Автосохранение не срабатывает

**Решение:**
1. Убедитесь что прошло минимум 30 секунд
2. Проверьте консоль браузера на ошибки
3. Проверьте что WebSocket соединение открыто (вкладка Network в DevTools)

### ❌ Красная ошибка при подключении WebSocket

**Причина:** WebSocket сервер слушает другой порт

**Решение:**
1. Проверьте `client.js`, строка ~91:
   ```javascript
   const WS_SERVER_URL = window.location.hostname === 'localhost' 
     ? `ws://localhost:3001`
     : `wss://${window.location.host}`;
   ```
2. Убедитесь что `websocket-server.js` запущен на порту 3001

## 📝 Логирование

### Включение полного логирования

Отредактируйте `client.js` и `websocket-server.js`, раскомментируйте `console.log` строки.

### Выключение логирования

Замените все `console.log` на `if (DEBUG) console.log`, добавьте флаг:
```javascript
const DEBUG = false; // true для разработки
```

## 🎯 Checklist перед продакшеном

- [ ] Пароль хешируется `bcrypt` а не Base64
- [ ] SSL/TLS сертификаты установлены (wss://)
- [ ] БД резервируется каждый день
- [ ] Логирование ошибок настроено
- [ ] Лимиты на частоту запросов установлены
- [ ] Валидация input на сервере добавлена
- [ ] CORS настроен если нужен
- [ ] Rate limiting для login попыток

## 📞 Что делать если ничего не работает

1. **Перезагрузите оба сервера** (Ctrl+C и запустите заново)
2. **Очистите браузер:**
   - DevTools → Application → Clear storage → Clear all
   - Удалите localStorage
3. **Проверьте консоль браузера** (F12 → Console)
4. **Проверьте консоль Node.js** на ошибки
5. **Очистите БД** если нужно:
   ```bash
   echo '{"accounts":{},"players":{},"clans":{},"leaderboard":[],"battles":{},"stats":{"totalBattles":0,"totalClans":0,"totalPlayers":0},"event":{}}' > database.json
   ```
