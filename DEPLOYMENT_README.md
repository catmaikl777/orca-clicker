## 📞 ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

### Q: Сколько это стоит?

**A:** Совершенно бесплатно!
- Render Free Tier (бэкенд) = $0
- Vercel Free Tier (фронтенд) = $0

### Q: Мои данные будут сохраняться?

**A:** Да! Система аккаунтов полностью работает:
- Регистрация и вход
- Автосохранение каждые 30 сек
- Восстановление при F5
- Долгосрочное хранилище в database.json

### Q: Как часто я могу обновлять игру?

**A:** Столько раз сколько хотите! Просто `git push`:
```bash
# Измените код
git add .
git commit -m "Описание изменений"
git push origin main
# Автоматический деплоймент за ~1 минуту
```

### Q: Render сервис спит?

**A:** Да, бесплатный план спит после 15 мин неактивности.
- Первый запрос может быть медленным (~30 сек)
- Решение: Платный план ($7/месяц) для continuous running

### Q: Как добавить больше игроков?

**A:** Система уже готова для множество игроков!
- Каждый игрок имеет свой аккаунт
- WebSocket обрабатывает сотни одновременных подключений
- Если нужно масштабировать: upgrade на Pro платы Render/Vercel

## 🆘 ПРОБЛЕМЫ?

### Проверьте логи:

**Render:**
```

```

**Vercel:**
```
Vercel Dashboard → orca-clicker → Deployments → Logs
```

### Comuni проблемы:

| Проблема | Решение |
|----------|---------|
| "Не подключается" | Проверьте VITE_WS_URL в Vercel |
| "WebSocket failed" | Проверьте FRONTEND_URL в Render |
| "Данные не сохраняются" | Проверьте логи Render |
| "Первый запрос медленный" | Это нормально (Render спит) |
