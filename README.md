## 🔧 Установка и запуск

### Требования
- Node.js (v14+)
- npm

### Шаги

1. **Установка зависимостей**
```bash
npm install
```

2. **Запуск WebSocket сервера** (Терминал 1)
```bash
node websocket-server.js
```
Ожидаемый вывод:
```
WebSocket сервер запущен на порту 3001
База данных загружена
```

3. **Запуск HTTP сервера** (Терминал 2)
```bash
node server.js
```
Ожидаемый вывод:
```
HTTP сервер запущен на http://localhost:3000
```

4. **Открыть в браузере**
```
http://localhost:3000
```

## 🧪 Проверка работы

### Быстрый тест (2 минуты)
1. Создайте аккаунт (нажмите "Регистрация")
2. Поиграйте (нажмите на косатку 5 раз)
3. Нажмите F5 (перезагрузка)
4. Войдите с тем же паролем
5. **Проверьте что монеты остались** ✅

Если монеты сохранились - система работает!

## 💡 Советы

1. Сначала покупайте авто-доход для пассивного заработка
2. Участвуйте в PvP баттлах для бонусных косаток
3. Собирайте бонусы, когда они появляются
4. Выполняйте квесты для дополнительных наград
5. Копите на киберпанк скин - он самый стильный!

## 🔗 Онлайн функционал

Для работы онлайн функций:
1. Запустите `websocket-server.js` на сервере
2. Откройте порт 3001 для WebSocket соединений
3. В `client.js` укажите адрес сервера
